    1. 安装环境

    1.1. 软件版本

    • Windows 11
    • Python 3.12.0
    • node v20.10.0
    • npm 10.2.3
    • git version 2.43.0.windows.1
    • typora
    • vscode
    • hexo-cli: 4.3.1

    1.2. 软件安装

    可以一直下一步。建议百度。

    2. 博客搭建

    2.1. Git配置

    在cmd打开。

    # 配置Git的用户、邮箱
git config --global user.name "xxx"
git config --global user.email "xxxxx@gmail.com
git config --list --global
# 配置GitHub的ssh密钥
ssh-keygen -t rsa -C "xxxxx@gmail.com"

    一直回车，记住保存的路径，一般是C:\Users\用户/.ssh/id_rsa

    ssh密钥在pub文件里，需将里面的内容拷贝到github里。

    打开https://github.com/settings/keys网页，SSH keys – new SSH Key。选Authentication keys。

    # 测试连通性
ssh -T git@github.com
# 输入yes
# 可以看到如下结果：
You've successfully authenticated, but GitHub does not provide shell access.

    Git就配置完成。

    2.2. GitHub配置

    2.2.1. 搭建图床

    创建图床repository。

    • 打开https://github.com/。

    • 点击GitHub首页右上角的“+”，点击New repository。

    • 在Repository name输入，tuchuang，拉到最下面创建。

    创建GitHub token。

    • 打开https://github.com/settings/tokens页面，点击Personal access tokens (classic)，创建。

    • note写piclist-hexo，知道自己干嘛用的。

    • Expiration选No expiration，永不过期。

    • Select scopes仅勾选repo。

    • 然后最下面创建，将token复制到piclist。

    配置piclist。

    • 下载piclist。https://piclist.cn/。

    • 打开piclist，点击图床 – GitHub。根据下图进行配置。

    • 配置完成后，在图床 – GitHub，点击设为默认图床。根据下图进行配置。

    配置typora。

    • 文件 – 偏好设置 – 图像。根据下图进行配置。

    2.2.2. 创建GitHub Page

    创建博客repository。

    • 打开https://github.com/。

    • 点击GitHub首页右上角的“+”，点击New repository。

    • 在Repository name输入，你的GitHub用户名.github.io，拉到最下面创建。

    2.3. hexo配置

    2.3.1. hexo初始配置

    选择一个目录，新建一个文件夹，在这里打开cmd。你也可以cd过去。

    # 安装hexo
npm install -g hexo-cli
# 查看版本
hexo -v
# 新建博客文件夹
mkdir GithubPage
# 进入文件夹
cd GithubPage
# 使用hexo初始化该文件夹作为博客
hexo init
# 安装依赖
npm install
# 创建静态页面
hexo g
# 打开web服务
hexo s
# 使用浏览器打开http://localhost:4000/

    hexo显示成功，则hexo初始化配置完成。

    2.3.2. hexo主题配置

    我这里重新搭，主要是修改根目录_config.yml文件和themes文件夹。

    每次修改都要运行命令：hexo clean && hexo g && hexo s。

    参考CSDN博客。一篇很用心的博客。

    https://blog.csdn.net/cat_bayi/article/details/128725230

    这里我使用了3-hexo主题，我觉得能够更好的展示与梳理知识体系。

    # 3-hexo下载安装
https://yelog.org/2017/03/07/3-hexo/
# 3-hexo使用说明
https://yelog.org/2017/03/23/3-hexo-instruction/

    首先下载主题，然后根据3-hexo的使用说明进行修改。

    篇幅所限，CSDN博客已经很详细了。一些需要关注的内容如下，可在使用说明找到。
