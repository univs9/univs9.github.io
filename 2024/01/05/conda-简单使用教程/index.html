    1. 官网下载

    https://www.anaconda.com/download

    安装的话，应该是可以直接下一步。

    知乎教程如下：

    https://zhuanlan.zhihu.com/p/524017924

    2. 配置环境

    参考教材如下：

    https://zhuanlan.zhihu.com/p/646201940

    在环境变量里添加D:\anaconda3和D:\anaconda3\Scripts。

    3. 使用VScode

    使用管理员方式打开Windows PowerShell。输入命令更改执行策略。

    Set-ExecutionPolicy RemoteSigned

    输入”Y”并按Enter键即可。

    4. 常用命令

    • 激活环境：激活一个环境。
    conda activate 环境名称
    • 退出环境：退出当前环境并返回到基础环境
    conda deactivate 环境名称
    • 查看所有安装的包：查看所有已安装的包。
    conda list
    • 查看特定包：查看特定包是否已安装。
    conda list 包名称
    • 安装包：安装一个包。
    conda install 包名称
    • 卸载包：卸载一个包。
    conda uninstall 包名称
    • 查看环境：查看当前存在的所有环境。
    conda env list
    • 创建环境：创建一个新的环境。
    conda create --name 环境名称
    • 删除环境：删除一个环境及其所有内容。
    conda remove -n 环境名称 --all
    • 配置清华源：配置conda使用清华源进行软件包下载。这可以加速某些包的下载速度。
    conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/   
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r/   
conda config --set channel_priority strict   
conda config --set show_channel_urls yes
