    2. 简单使用

    2.1. 镜像与tag

    首先要有一个镜像，使用docker pull xxx:tags，下载镜像。

    以ubantu为例，你首先是有要有一个镜像，可以在hub.docker.com里寻找，并可能需要选择相应的tag。

    然后使用docker pull ubantu拉取镜像。可以看到第一行信息Using default tag: latest，默认使用latest的tag。

    下载了镜像，就可以根据自己的需求对容器进行创建。

    2.2. 常用命令

    • docker search 镜像名称

      从docker hub寻找镜像，常用参数--no-trunc，不打断描述。

    • docker pull 镜像名称

      拉取镜像，可以指定某个tag，如docker pull ubuntu:22.04。

    • docker images 

      查看镜像。常用参数-q，仅镜像id。

    • docker run 镜像名称

      用一个镜像创建一个新的容器。-p设置端口映射。-it开启容器交互并启动tty终端。-d容器后台运行。-name为容器命名。

    • docker ps 

      查看容器。常用-a，默认仅显示运行的容器。

    • docker start 容器ID或容器名

      启动一个容器。

docker stop、restart 容器ID或容器名。

    • docker exec -it 容器ID或容器名 /bin/bash

      在容器执行命令。需要配合-it，才能有终端交互。

    • docker update 容器ID或容器名

      修改容器的配置。如修改cpu、内存。

自动重启策略docker update --restart=always 容器ID或容器名。

    • docker rm 容器ID或容器名

      删除容器。

    • docker rmi 镜像名称:镜像tag/镜像Id

      删除镜像。

    2.3. 实践案例

    示例，这里用tomcat，快点。

      ①拉取tomcat镜像
	z@hp:~$ sudo docker pull tomcat
	Using default tag: latest
	latest: Pulling from library/tomcat
  
  ②查看镜像
	z@hp:~$ sudo docker images
	REPOSITORY       TAG       IMAGE ID       CREATED        SIZE
	tomcat           latest    3db0f5668a77   2 days ago     460MB
  
  ③从镜像创建新的容器，-d让容器后台运行，-p指定端口映射8080，--name指定容器名字zzz，最后一个参数指定镜像。
	sudo docker run -d -p 8080:8080 --name zzz tomcat
  
  ④查看容器状态
	z@hp:~$ sudo docker ps 
	CONTAINER ID   IMAGE     COMMAND             CREATED              STATUS              PORTS                                       NAMES
	4102adb4ed3c   tomcat    "catalina.sh run"   About a minute ago   Up About a minute   0.0.0.0:8080->8080/tcp, :::8080->8080/tcp   zzz
 
 ⑤使用浏览器可以访问。
 
 ⑥进入容器交互模式，执行命令。
	z@hp:~$ sudo docker exec -it zzz /bin/bash
	root@4102adb4ed3c:/usr/local/tomcat# ls
		bin conf lib logs NOTICE RELEASE-NOTES  .......
	使用exit退出。
  
  ⑦修改容器参数，使容器自动重启
	z@hp:~$ sudo docker update --restart always zzz
zzz

    3. 安装docker

    3.1. Windows安装

    在WSL下安装，需要Windows 10 21H2的新特性，用linux子系统，暂不深究。

    需要安装桌面版docker，下载链接如下：

    https://www.docker.com/products/docker-desktop/

    您的WSL可能版本较旧，可能需要更新WSL，使用wsl –update进行更新，在cmd界面里敲就行。

    另，Windows使用WSL，可能需要hyper-v，开启hyper-v，与VMware workstation发生冲突，若需要使用VMware，则需要关闭hyper-v，命令：bcdedit /set hypervisorlaunchtype off ，并在控制面板 - 程序 - Windows功能处关闭Linux子系统和虚拟机平台。

    3.2. Linux安装

    以下是ubantu安装dokcer的命令。

    sudo apt install docker.io
sudo systemctl enable docker
sudo docker version
